# Body and Syntax

The **Body** section is where you'll enter the contents of your post.

Our text editor uses Markdown. For a detailed guide on Markdown, click [here](https://www.markdownguide.org/basic-syntax/).

If you're new to Markdown, we have a few options along the top of the text box to make things easier.

1. Click the "B" to bold text
2. Click the "I" to italicize text
3. Click the quotation marks to turn text into a block quote
4. Click the list icon to create a bulleted list
5. Click the numbered list icon to create a numbered list
6. Click the chain icon to create a hyperlink
