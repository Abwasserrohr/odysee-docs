# Login Links

If you'd rather login using an email link than a password, you can select **Use Magic Link**.

![](https://i.imgur.com/sxATuIE.png)

This sends an confirmation email. Simply click the button in this email and complete a CAPTCHA, and you'll be logged in.

![](https://i.imgur.com/MttvyYb.png)

**Note: if you receive this email but weren't trying to log in, don't click the link.**
