# Republication

Vous avez vu une vidéo tellement géniale que vous aimeriez la partager pour que d'autres la voient ? Vous pouvez le faire en la *republiant*.

Le fait de republir une vidéo la fera apparaître sur la page de votre chaîne, sous *Contenu*.

Le fait de republier une vidéo peut, en fonction du montant investi, augmenter la visibilité de la vidéo sur Odysee. Pour plus d'informations à ce sujet, consultez notre section sur la [dénomination](https://help.odysee.tv/category-blockchain/category-staking/naming).
