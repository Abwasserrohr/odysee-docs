# Ajouter une méthode de paiement

En ajoutant une carte de crédit ou de débit, vous pourrez donner des pourboires en espèces aux créateurs, précommander des vidéos, et plus encore.

Pour ajouter une carte, vous devez d'abord ouvrir votre porte-monnaie. Ce dernier se trouve en haut à droite de l'écran, dans la barre de menu.

![](https://i.imgur.com/RUtD2EO.jpg)

Sur la version mobile, vous le trouverez en bas du menu latéral.

![](https://i.imgur.com/7Vq2b1a.png)

De là, vous sélectionnerez **Méthodes de paiement**.

![](https://i.imgur.com/Ph24glk.png)

Sur l'écran Méthodes de paiement, il vous suffit de saisir votre nom complet, le numéro de votre carte, sa date d'expiration et son code CVC à 3 chiffres.

![](https://i.imgur.com/rmYEuMt.png)

Si vous avez déjà ajouté une carte, elle apparaîtra sur l'écran Méthodes de paiement.

![](https://i.imgur.com/pUuzmPC.png)

À partir de là, vous pouvez **supprimer la carte**, **consulter l'historique de vos transactions**, et **changer votre devise de USD en Euros, ou vice versa**.
