# Liens de connexion

Si vous préférez vous connecter en utilisant un lien électronique plutôt qu'un mot de passe, vous pouvez sélectionner **Utiliser le lien magique**.

![](/img/fr/Liendeconnexion.png)

Cela envoie un courriel de confirmation. Il suffit de cliquer sur le bouton dans ce courriel et de compléter un CAPTCHA, et vous serez connecté.

![](https://i.imgur.com/MttvyYb.png)

**Note : si vous recevez ce courriel mais que vous n'avez pas essayé de vous connecter, ne cliquez pas sur le lien.**

## Pourquoi me demande-t-on de vérifier par un lien ?

Parfois, lorsque vous vous connectez, il se peut que vous receviez un courriel de confirmation sans l'avoir demandé. Cela se produit lorsque nous détectons un changement d'IP. La raison de sécurité derrière cela est d'empêcher les personnes autre que vous de se connecter à votre compte, même si elles connaissent votre nom d'utilisateur et votre mot de passe.

Si vous changez de lieu entre deux connexions, il vous sera demandé de confirmer par courriel, même si vous avez déjà utilisé Odysee à ces deux endroits.

Cela peut également être enclenché si vous utilisez un VPN.
