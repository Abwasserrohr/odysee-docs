# Se connecter à Odysee

Vous avez déjà un compte et vous revenez sur Odysee ? Vous cliquerez sur le bouton **Connexion** dans le coin supérieur droit.

![](https://i.imgur.com/BVk7SoC.jpg)

Sur un mobile, vous cliquerez sur l'icône de la personne, ce qui affichera Connexion dans le menu déroulant.

![](https://i.imgur.com/vWURDEo.jpg)

Vous commencerez par saisir votre adresse électronique.

![](https://i.imgur.com/vpHKwo4.png)

Sur l'écran du mot de passe, vous avez quelques options.

![](https://i.imgur.com/sxATuIE.png)

Si vous connaissez votre mot de passe et que vous souhaitez vous connecter, il vous suffit de le saisir et de cliquer sur Continuer.
