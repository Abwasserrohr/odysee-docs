# Changement de mot de passe

Lorsque vous êtes connecté à votre compte, vous pouvez changer votre mot de passe dans le menu **Paramètres**. 

Cliquez sur l'icône de votre chaîne dans le coin supérieur droit, puis sélectionnez **Paramètres** dans le menu déroulant.

![](/img/fr/Changementdemotdepasse01.png)

# Changement de mot de passe
À partir de là, vous ferez défiler l'écran jusqu'à **Compte** et cliquerez sur **Gestion** dans l'onglet **Mot de passe**.

Ensuite, il vous suffit d'entrer votre ancien mot de passe, ce que vous voulez que votre nouveau mot de passe soit, et cliquez sur **Définir le mot de passe**.

![](/img/fr/Changementdemotdepasse02.png)
