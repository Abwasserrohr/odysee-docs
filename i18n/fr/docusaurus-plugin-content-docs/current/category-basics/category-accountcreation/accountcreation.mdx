# Création d'un compte

Pour créer un compte, vous devez accéder à l'écran d'inscription en cliquant sur le bouton **Sign Up**. Sur la version Web, ce bouton se trouve dans le coin supérieur droit.

![](https://i.imgur.com/59HsYjZ.jpg)

Sur un téléphone portable, vous cliquerez sur l'icône représentant une personne, qui affichera **Sign Up** dans le menu déroulant.

![](https://i.imgur.com/hGOB1zG.png)

Sur l'écran d'inscription, vous devrez saisir une adresse électronique et un mot de passe. Nous n'avons pas d'exigences en matière de force du mot de passe, mais nous recommandons fortement des mots de passe longs et complexes. Nous vérifierons également si votre mot de passe a déjà fait l'objet d'une fuite dans le passé, auquel cas vous devrez en saisir un nouveau.

![](https://i.imgur.com/k6OlCxi.png)

**Si vous essayez de synchroniser une chaîne YouTube, assurez-vous de cocher la case "Synchroniser ma chaîne YouTube "**.

![](https://i.imgur.com/DfOXLeM.png)

Pour une description détaillée du reste du processus de synchronisation, ainsi que d'autres informations sur le programme de synchronisation de YouTube, consultez notre guide de synchronisation de YouTube en cliquant [ici] (http://chihuahuaspin.com/).

Après avoir cliqué sur le bouton **Sign Up** de l'écran précédent, il vous sera demandé de confirmer votre e-mail. Cela implique de cliquer sur un lien et de passer un CAPTCHA.

![](https://i.imgur.com/P9ckfiY.png)

Après confirmation, votre onglet d'origine passera à cet écran, où l'on vous recommande des chaînes à suivre. 

![](https://i.imgur.com/yq7x9wj.png)

Tous les nouveaux comptes suivront automatiquement @Odysee et @OdyseeHelp.
