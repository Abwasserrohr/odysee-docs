# Changement d'adresse électronique

Pour l'instant, nous ne disposons pas d'un moyen intégré de modifier les adresses électroniques. Nous pouvons le faire, mais nous devons le faire manuellement.

Si vous souhaitez que votre adresse électronique soit modifiée, veuillez envoyer un courriel à hello@odysee.com.
