# Modification des téléchargements

Pour modifier un téléchargement, vous pouvez cliquer sur le bouton de modification sur le téléchargement lui-même:

Ou vous pouvez cliquer sur le bouton d'édition dans le menu à trois points sans cliquer sur le téléchargement:

À partir de là, vous pouvez modifier tout ce qui concerne le téléchargement, sauf l'URL. Cela inclut la modification du fichier. Ainsi, par exemple, si vous avez créé une vidéo et que vous n'avez remarqué qu'après le téléchargement qu'il y avait une erreur, vous pouvez simplement changer le fichier téléchargé en un fichier fixe plutôt que de devoir supprimer entièrement le téléchargement et recommencer.

Une fois vos modifications effectuées, veillez à cliquer sur **Sauvegarder** en bas de l'écran d'édition.
