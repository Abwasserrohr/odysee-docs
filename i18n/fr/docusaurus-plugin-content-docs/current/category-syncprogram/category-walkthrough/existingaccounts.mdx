# Synchronisation avec un compte existant

Pour synchroniser une chaîne YouTube avec un compte Odysee existant, vous devez d'abord cliquer sur l'icône de la personne dans le coin supérieur droit (si vous avez déjà une chaîne, cette icône sera l'image de profil de votre chaîne), puis cliquer sur **Chaînes**.

![](/img/fr/Synchronisationcompteexistant01.png)

Dans cet écran, vous sélectionnerez **Synchroniser la chaîne YouTube**.

![](/img/fr/Synchronisationcompteexistant02.png)
