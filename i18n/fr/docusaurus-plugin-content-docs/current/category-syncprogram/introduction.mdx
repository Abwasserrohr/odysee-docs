# Introduction

Télécharger des centaines d'anciennes vidéos sur une nouvelle plateforme peut être une tâche ardue. Heureusement, Odysee dispose d'un outil de synchronisation YouTube qui permet aux créateurs de YouTube de mettre automatiquement en miroir tout le contenu passé et futur sur leur chaîne Odysee.

Dans cet article, nous allons passer en revue les principes de base de l'outil de synchronisation elle-même, fournir une description détaillée et répondre à quelques questions fréquemment posées.
