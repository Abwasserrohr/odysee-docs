# Créer plus de chaînes

Pour créer une autre chaîne, vous devez d'abord retourner à la page **Chaînes**.

![](/img/fr/Synchronisationcompteexistant01.png)

À partir de cette page, vous pourrez créer d'autres chaînes en utilisant le bouton **Nouvelle chaîne**.

![](/img/fr/Créerdenouvelleschaines.png)

Vous pouvez avoir jusqu'à 5 chaînes.
