# Was macht Odysee mit meinem YouTube-Kanal?

Die Synchronisierung ist eine sogenannte SQL-Abfrage, dabei werden nur Metadaten abgegriffen. Es handelt sich dabei auch nur um einen Lesezugriff. Im Klartext bedeutet das: Die Daten werden nur einmalig abgegriffen. Wenn du zum Beispiel später den Text oder das Thumbnail bei YouTube änderst, wird diese Änderung nicht automatisch auf Odysee übernommen, da es sich um eine einmalige Abfrage handelt.
