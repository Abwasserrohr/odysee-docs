# YouTube-Sync mit Konto einrichten

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="web" label="Web" default>
    Um einen YouTube-Kanal mit einem bestehenden Odysee-Konto zu synchronisieren, klicke zuerst auf das Personensymbol in der oberen rechten Ecke (wenn du bereits einen Kanal hast, ist dieses Symbol das Profilbild deines Kanals) und klicke auf <b><a href="https://odysee.com/$/channels">YouTube-Kanal synchronisieren</a></b>.
