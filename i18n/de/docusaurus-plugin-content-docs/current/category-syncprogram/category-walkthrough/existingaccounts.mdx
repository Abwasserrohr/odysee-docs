# YouTube-Sync mit Konto einrichten

Um einen YouTube-Kanal mit einem bestehenden Odysee-Konto zu synchronisieren, klicke zuerst auf das Personensymbol in der oberen rechten Ecke (wenn du bereits einen Kanal hast, ist dieses Symbol das Profilbild deines Kanals) und klicke auf **[Kanäle](https://odysee.com/$/channels)** .

![](https://i.imgur.com/KB8nfF3.png)

Wähle auf diesem Bildschirm **[YouTube-Kanal synchronisieren](https://odysee.com/$/youtube?reset_scroll=youtube&new_channel=true)** aus.

![](https://i.imgur.com/sBAt3IC.png)
