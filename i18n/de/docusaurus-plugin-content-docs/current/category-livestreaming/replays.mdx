# Livestreams als Video-On-Demand veröffentlichen

Zurzeit gibt es keine Option zur automatischen Veröffentlichung von vergangenen Livestreams als Video-On-Demand (VOD). Hier zeigen wir dir, wie du den Livestream als VOD manuell einrichten kannst. 

Um einen vergangenen Livestream als VOD zu veröffentlichen, kehre zum Bildschirm **Veröffentlichen** unter der Registerkarte *Live gehen* zurück.

*image*

Von hier aus wähle **Wählen Sie eine Wiedergabe** aus. Falls keine Wiederholungen angezeigt werden, kannst du die Seite aktualisieren, indem du **Nach Wiederholungen suchen** auswählst. 

Wähle im Menü die Livestream-Wiedergabe aus, die du veröffentlichen möchtest, gebe die erforderlichen Informationen ein und wähle unten auf der Seite **Erstellen**.
