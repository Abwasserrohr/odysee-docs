# Livestream richtig einstellen 

Nachdem dein Livestream erstellt wurde, wechsel zum Tab **[Lokale Einrichtung](https://odysee.com/$/livestream?t=Setup)**.

![](/img/de/stream_key_and_stream_server_screen_dark.png#dark)
![](/img/de/stream_key_and_stream_server_screen_light.png#light)

Der Streamserver und der Streamschlüssel werden verwendet, um deiner Streaming-Software mitzuteilen, wohin dein Livestream gesendet werden soll. Kopiere diese Angaben und gebe sie bei Bedarf in die Streaming-Software ein.

## OBS

OBS: `Datei` > `Einstellungen` > `Stream`

**Service**: *Benutzerdefiniert* auswählen

**Server**: gebe den auf Odysee bereitgestellten Streamserver ein

**Stream Key**: gebe den auf Odysee bereitgestellten Streamschlüssel ein

Sofern alles korrekt ist, klicke auf **Okay**.

## Streamyard

*Hinweis: Um mit Streamyard zu Odysee zu streamen, wird die kostenpflichtige Version von Streamyard benötigt.*

Auf dem Streamyard-Hauptbildschirm wähle nach dem Einloggen auf der rechten Seite **Destinations**.

*image*

Dann **Add Destination** and **Custom RTMP** auswählen.

Auf dem nächsten Bildschirm beginne mit der Eingabe deiner Stream-Information.

Unter **RTMP server URL**, gebe den auf Odysee bereitgestellten Streamserver ein.

Unter **Stream Key**, gebe den auf Odysee bereitgestellten Streamschlüssel ein.

Wenn alles korrekt ist, klicke auf **Add RTMP Server**.


## Restream

*Um mit Restream zu Odysee zu streamen, benötigst du die kostenpflichtige Version von Restream.*

*image*

Auf dem Restream-Hauptbildschirm wähle nach dem Einloggen **Stream With Studio**.

*image*

Sobald du im Studio bist, wähle **Add Destination** und **Custom RTMP**.

*image*

Auf dem nächsten Bildschirm beginne mit der Eingabe deiner Stream-Information.

Unter **RTMP server URL**, gebe den auf Odysee bereitgestellten Streamserver ein.

Unter **Stream Key**, gebe den auf Odysee bereitgestellten Streamschlüssel ein.

Wenn alles korrekt ist, klicke auf **Add Channel**.
