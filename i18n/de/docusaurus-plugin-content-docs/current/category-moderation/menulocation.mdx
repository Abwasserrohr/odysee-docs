# Wo finde ich die Moderationswerkzeuge? 

Das Menü für die Einstellungen kannst du über das Profilbild in der oberen rechten Ecke anklicken und im Dropdown-Menü **Einstellungen** auswählen.

*image*

Auf mobilen Geräten befindet sich das Einstellungsmenü am unteren Rand der erweiterten Seitenleiste.

*image*

Im Einstellungsmenü **Inhaltseinstellungen** auswählen und dann auf **Kanaleinstellungen** klicken.

*image*
