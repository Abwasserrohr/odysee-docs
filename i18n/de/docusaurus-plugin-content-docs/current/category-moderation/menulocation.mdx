# Wo finde ich die Moderationswerkzeuge? 

Das Menü für die Einstellungen kannst du über das Profilbild in der oberen rechten Ecke anklicken und im Dropdown-Menü [<OdyseeIcon type="cogwheel"/> Einstellungen](https://odysee.com/$/settings) auswählen.

<Image src="/img/de/locate_settings_menu_desktop_dark.png#dark" />
<Image src="/img/de/locate_settings_menu_desktop_light.png#light" />

Auf mobilen Geräten befindet sich das Einstellungsmenü weiter unten in der erweiterten Seitenleiste.

<Image src="/img/de/locate_settings_menu_mobile_dark.png#dark" />
<Image src="/img/de/locate_settings_menu_mobile_light.png#light" />

Im Einstellungsmenü unter **Inhaltseinstellungen** findest du die **Kanaleinstellungen**, klicke darunter auf [<OdyseeIcon type="ChevronRight"/> Verwalten](https://odysee.com/$/settings/creator).

<Image src="/img/de/locate_channel_settings_dark.png#dark" />
<Image src="/img/de/locate_channel_settings_light.png#light" />
