# Hochladen und Dateiauswahl

Odysee ist vor allem als Videoplattform bekannt, aber du kannst auch Bilder, Audiodateien, PDFs, CAD-Dateien und so ziemlich alles andere hochladen.

Auf dem Upload-Bildschirm wirst du zunächst aufgefordert, eine Datei auszuwählen.

Für Videos müssen Sie MP4-Dateien im Format H264/AAC hochladen. Für eine optimale Anzeige empfehlen wir Bitraten unter 8 Mbps.

:::info Hinweis
Odysee-Uploads sind auf 16 GB beschränkt.
:::
