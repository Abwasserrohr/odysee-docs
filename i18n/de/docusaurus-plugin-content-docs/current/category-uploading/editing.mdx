# Uploads bearbeiten

Um einen Upload zu bearbeiten, kannst du entweder die Schaltfläche "Bearbeiten" im Upload klicken oder auf die Schaltfläche "Bearbeiten" im Drei-Punkte-Menü klicken, ohne den Upload anzuklicken:

Von hier aus kannst du alle Daten des Uploads außer der URL bearbeiten. Dazu gehört auch das Ändern der Datei. Wenn du z. B. ein Video erstellt hast und erst nach dem Hochladen bemerkst, dass ein Fehler vorliegt, können Sie die hochgeladene Datei einfach in eine *fixed* Datei ändern, anstatt den Upload komplett zu löschen und von vorne zu beginnen.

Sobald alle Änderungen vorgenommen wurde, auf **Speichern** klicken.
