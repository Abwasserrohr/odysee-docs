# Auf Odysee einloggen

Du hast bereits ein Konto und möchtest dich bei Odysee einloggen? Klicke oben rechts auf die Schaltfläche **[Anmelden](https://odysee.com/$/signin)**.

![](https://i.imgur.com/BVk7SoC.jpg)

Auf dem Handy drückst du auf das Personensymbol, das im Dropdown-Menü **Anmelden** anzeigt.

![](https://i.imgur.com/vWURDEo.jpg)

Gib deine E-Mail-Adresse und dannach dein Passwort ein.

![](https://i.imgur.com/vpHKwo4.png)

![](https://i.imgur.com/sxATuIE.png)

Wenn du dein Passwort kennst und dich anmelden möchtest, gib es einfach ein und klicke auf **Weiter**. Alternativ kannst du dir auch einen Magic-Link über deine E-Mail-Adresse zuschicken lassen.
