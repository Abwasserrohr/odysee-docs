# Auf Odysee einloggen

Du hast bereits ein Konto und kommst zu Odysee zurück? Klicke oben rechts auf die Schaltfläche **[Anmelden](https://odysee.com/$/signin)**.

![](https://i.imgur.com/BVk7SoC.jpg)

Auf dem Handy drückst du auf das Personensymbol, das im Dropdown-Menü Anmelden anzeigt.

![](https://i.imgur.com/vWURDEo.jpg)

Beginnen mit der Eingabe deiner E-Mail-Adresse.

![](https://i.imgur.com/vpHKwo4.png)

Auf dem Passwortbildschirm hast du weitere Optionen.

![](https://i.imgur.com/sxATuIE.png)

Wenn du dein Passwort kennst und dich anmelden möchtest, gib es einfach ein und klicke auf **Weiter**. Alternativ kannst du dir auch einen Magic Login Link über E-Mail zusenden lassen, mit dem du dich ebenfalls einloggen kannst.
