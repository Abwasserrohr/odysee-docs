# Kanäle bearbeiten

Um deinen Kanal zu bearbeiten, besuche deinen Kanal und klicke auf das das Stiftsymbol unten rechts in deinem Kanalbanner.

![](https://i.imgur.com/ogPLyIt.png)

Wenn du deinen Kanal fertig bearbeitet hast, speichere alle Änderungen, damit diese übernommen werden.

![](https://i.imgur.com/ZISLVpY.png)
