# Kanäle löschen

Um einen Kanal zu löschen, hast du einige Möglichkeiten.


Du kannst Kanäle auf der **[Kanäle](https://odysee.com/$/channels)** Seite mithilfe des Drei-Punkte-Menüs löschen.

![](https://i.imgur.com/SLP1Axa.png)

Du kkannst einen Kanal auch löschen, indem du das Drei-Punkte-Menü auf dem Kanal selbst verwendest.


![](https://i.imgur.com/CNcFzOy.png)

Wenn du dich entscheidest, einen Kanal mitten in der Bearbeitung zu löschen, gibt es auch eine Löschoption im Bearbeitungsmenü.

![](https://i.imgur.com/7F8uSTA.png)
