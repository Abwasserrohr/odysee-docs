# Weitere Kanäle erstellen

Um einen weiteren Kanal zu erstellen, kehre zunächst zur Seite **[Kanäle](https://odysee.com/$/channels)** zurück.

![](/img/de/homepage_profile_menu_open_arrow_to_channels_desktop_dark.png#dark)
![](/img/de/homepage_profile_menu_open_arrow_to_channels_desktop_light.png#light)


Auf der Kanalübersicht kannst du weitere Kanäle mit der **[Neuer Kanal](https://odysee.com/$/channel/new)** Schaltfläche erstellen.

![](/img/de/sync_more_channels_arrow_dark.png#dark)
![](/img/de/sync_more_channels_arrow_light.png#light)

Du kannst bis zu fünf Kanäle erstellen.
