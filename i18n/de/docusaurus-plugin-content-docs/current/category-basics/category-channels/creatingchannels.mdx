# Weitere Kanäle erstellen

Um einen weiteren Kanal zu erstellen, kehre zunächst zur Seite **[Kanäle](https://odysee.com/$/channels)** zurück.

![](https://i.imgur.com/KB8nfF3.png)

Auf der Kanalübersiucht kannst du weitere Kanäle mit der **[Neuer Kanal](https://odysee.com/$/channel/new)** Schaltfläche erstellen.

![](https://i.imgur.com/5ygxxOS.png)

Du kannst bis zu 5 Kanäle haben.
