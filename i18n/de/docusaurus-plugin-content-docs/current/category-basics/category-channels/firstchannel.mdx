# Kanal erstellen

Um Funktionen wie Likes, Dislikes, Kommentare und das Erstellen von Playlists zu nutzen, brauchst du zuerst einen Kanal.

Klicke oben rechts auf das Personensymbol (wenn du bereits einen Kanal hast, ist dieses Symbol das Profilbild deines Kanals) und klicke auf **Kanäle**.

![](https://i.imgur.com/KB8nfF3.png)

Auf der [Kanalübersicht](https://odysee.com/$/channels) kannst du auf **Kanal erstellen** klicken.

![](https://i.imgur.com/sq7vGiG.png)

Wenn du deinen YouTube-Kanal synchronisieren möchtest, kannst du auch auf **YouTube-Kanal synchronisieren** klicken, ohne ein neues Konto erstellen zu müssen.

:::info

Nach Abschluss kann die Erstellung des Kanals einige Minuten in Anspruch nehmen, bis dein neuer Kanal verfügbar ist.

:::
