# Verifizierung

Eine Verifizierung ist für die Verwendung von Odysee nicht erforderlich, wird jedoch dringend empfohlen. Durch die Verifizierung kannst du, während du Odysee nutzt, [LBRY Credits](/category-blockchain/overview) (kurz LBC) verdienen. Die Überprüfung ist notwendig, um Missbrauch zu vermeiden.

:::info

Willst du mehr über LBC und Odysee erfahren? Schau dir dazu gerne den [LBRY Credits-Leitfaden](/category-blockchain/) an.

:::

Die meisten Konten werden automatisch verifiziert. Abhängig von Faktoren wie Standort oder VPN-Nutzung können möglicherweise mehr Informationen für die Überprüfung notwendig sein. Dazu hast du einige Möglichkeiten:

![](/img/de/verify_to_earn_site_dark.png#dark)
![](/img/de/verify_to_earn_site_light.png#light)

### **Über Mobiltelefon**

Deine erste Option ist die Bestätigung über dein Telefon. Wir senden dir eine SMS mit einem Code um dein Konto zu bestätigen.

### **Mit einer Kreditkarte** 

Du kannst dich auch mit einer Kredit- oder Debitkarte verifizieren. Wir speichern deine Karteninformationen nicht und berechnen dir keine Gebühren. Möglicherweise siehst du eine Autorisierung in Höhe von einem Euro, diese wird aber nicht von uns eingelöst.

### **Auf Discord**

Schließlich kannst du dich über unseren Discord-Server verifizieren. Hier wird ein Moderator dein Konto überprüfen und dich, wenn du berechtigt bist, für Prämien freischalten. Unser Discord ist [hier](https://discord.com/invite/sWedwsNWjX) erreichbar.

Derzeit werden Verifizierungsanfragen manuell überprüft. Das bedeutet, dass es einige Stunden und in Ausnahmefällen auch mehreren Tagen dauern kann, bis die Verifizierung genehmigt wird.
