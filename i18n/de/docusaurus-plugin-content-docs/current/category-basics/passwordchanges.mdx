# Passwort ändern

Während du angemeldet bist, kannst du dein Passwort im Menü **[Einstellungen](https://odysee.com/$/settings)** ändern.

Klicke oben rechts auf dein Kanalsymbol und wähle im Dropdown-Menü **[Einstellungen](https://odysee.com/$/settings)** aus.

![](https://i.imgur.com/jnjajPu.jpg)

Von hier aus scrolle nach unten zu **Konto** und klicke auf **[Verwalten](https://odysee.com/$/settings/update_password)** in der Registerkarte **Passwort**.

Gib dort dein altes Passwort und das neue Passwort ein. Klicke danach auf **Passwort festlegen**, um das neue Passwort zu speichern.

![](https://i.imgur.com/OKtlKuY.png)
