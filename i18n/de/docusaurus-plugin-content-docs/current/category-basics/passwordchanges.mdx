# Passwort ändern

Während du angemeldet bist, kannst du dein Passwort im Menü **[<OdyseeIcon type="cogwheel" /> Einstellungen](https://odysee.com/$/settings)** ändern.

Klicke oben rechts auf dein Kanalsymbol und wähle im Dropdown-Menü **[<OdyseeIcon type="cogwheel" /> Einstellungen](https://odysee.com/$/settings)** aus.

![](/img/de/homepage_profile_menu_open_arrow_settings_desktop_light.png#light)
![](/img/de/homepage_profile_menu_open_arrow_settings_desktop_dark.png#dark)

Von hier aus scrolle nach unten zu **Konto** und klicke auf **[Verwalten](https://odysee.com/$/settings/update_password)** in der Registerkarte **Passwort**.

Gib dort dein altes Passwort und das neue Passwort ein. Klicke danach auf **Passwort festlegen**, um das neue Passwort zu speichern.

![](/img/de/change_password_desktop_light.png#light)
![](/img/de/change_password_desktop_dark.png#dark)
