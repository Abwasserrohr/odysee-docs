# E-Mail ändern

Derzeit haben wir keine integrierte Möglichkeit, E-Mails zu ändern. Dieser Vorgang muss manuell von uns durchgeführt werden.

Wenn du deine E-Mail ändern möchtest, sende uns bitte eine E-Mail an hello@odysee.com.
