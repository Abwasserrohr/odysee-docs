# Abonnieren

Gibt es einen kreativen Geist, dessen Inhalte du so richtig feierst? Du kannst ihm/ihr folgen, indem du auf Abonnieren klickst. 

Allen Kanälen, denen du folgst, werden unter **Meine Abos** angezeigt. Diese Seite findest du oben auf der Startseite oder in der Seitenleiste.

Unter **Meine Abos** kannst du auch die Schaltfläche **Kanäle entdecken** auswählen, um so neue Kanäle und Inhalte ausfindig zu machen.
