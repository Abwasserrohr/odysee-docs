# Abonnieren

Gibt es einen kreativen Odysee-Geist, dessen Inhalte du besonders magst? Du kannst ihm oder ihr folgen, in dem du auf die Schaltfläche **Abonnieren** auf ihrem Kanal oder unter ihren Inhalten klickst. 

![](/img/de/channel_page_subscribe_arrow_desktop_dark.png#dark)
![](/img/de/channel_page_subscribe_arrow_desktop_light.png#light)

Immer wenn du einen Kanal abonnierst, wird dieser unter **[Meine Abos](https://odysee.com/$/following)** hinzugefügt. Diese Kategorie wird in der Seitenleiste angezeigt.

Videos unter "Meine Abos" werden standardmäßig nach dem Veröffentlichunsdatum sortiert.
