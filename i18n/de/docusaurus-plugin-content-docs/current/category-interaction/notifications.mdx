# Benachrichtigungen

In der oberen rechten Ecke neben dem Profilbild befindet sich ein Glockensymbol. Hier kannst du deine Benachrichtigungen einsehen.

Es gibt eine Reihe von verfügbaren Benachrichtigungen, darunter:

- Kommentare unter deinen Videos
- Antworten auf deine Kommentare
- Trinkgeld-Benachrichtigungen
- Neue Abonnenten

Außerdem kannst du Benachrichtigungen für bestimmte Kanäle einstellen. 

Navigiere zur Kanalseite und klicke auf das Glockensymbol oben rechts im Banner. Mit der Aktivierung wirst du benachrichtigt, wenn neue Videos hochgeladen werden oder Kommentare außerhalb deren eigenen Kanals hinterlassen werden.
