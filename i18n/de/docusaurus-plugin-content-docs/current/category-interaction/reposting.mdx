# Reposten

Du hast ein so tolles Video gesehen, dass du es anderen zeigen möchten? Verwende dafür die Repost funktion.

Wenn Sie ein Video repostest, wird es direkt auf der Videoübersicht deines Kanals angezeigt.

Reposten eines Videos kann, je nach gesetztem Betrag, die Auffindbarkeit eines Videos auf Odysee erhöhen. Weitere Informationen hierzu findest du in unserem Abschnitt über die [Namensgebung](/category-blockchain/category-staking/naming).
