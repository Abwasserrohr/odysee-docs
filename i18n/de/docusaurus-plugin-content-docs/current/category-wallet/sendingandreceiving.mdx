# LBC empfangen und versenden

## Versenden

Um LBC von der Brieftasche aus zu versenden, klicke auf die Schaltfläche **Senden**, die unten zu finden ist.

Von hier aus kannst du eine direkte Adresse eingeben. Wenn du eine LBC-Brieftasche außerhalb von Odysee hast, kannst du die Adresse hier einfügen.

Alternativ kannst du auch andere kreative Geister innerhalb von Odysee suchen und mit LBC unterstützen.

## Empfangen

Um LBC zu empfangen, klicke auf die Schaltfläche **Empfangen**, die unten zu finden ist. 

Hier wird die Adresse deiner Brieftasche angezeigt. Wenn dir jemand LBC an diese Adresse sendet, erscheint es in deiner Brieftasche auf Odysee.
