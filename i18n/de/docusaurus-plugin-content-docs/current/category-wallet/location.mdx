# Wo finde ich meine Brieftasche? 

Dir ist bestimmt schon das LBC-Symbol <OdyseeIcon type="credits"/> und ein Zahlenwert in der Navigationsleiste aufgefallen. 

![](/img/de/locate_wallet_desktop_dark.png#dark)
![](/img/de/locate_wallet_desktop_light.png#light)

Das zeigt die Gesamtzahl deiner LBRY-Credits an. Wenn du draufklickst, kommst du zu deiner <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits"/> Brieftasche</OdyseeButton>.
