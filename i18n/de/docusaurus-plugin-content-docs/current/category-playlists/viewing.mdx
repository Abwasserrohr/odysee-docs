# Playlisten anzeigen 

## Deine Playlisten

Du kannst deine eigenen Playlisten anzeigen, indem du in der Seitenleiste auf die Schaltfläche **Playlisten** klickst.

Alle dein veröffentlichten und unveröffentlichten Wiedergabelisten werden in diesem Menü angezeigt.

## Andere Playlisten

Um die Playlisten eines anderen Kanals anzuzeigen, gehe zur entsprechenden Kanalseite. 

Wähle von hier aus die Registerkarte **Playlisten** aus. Alle öffentlichen Playlisten, die von diesem Kanal erstellt wurden, werden hier angezeigt. Alle Playlisten, die nicht öffentlich gemacht wurden, werden hier nicht angezeigt und sind nur für den Kanal sichtbar, der sie erstellt hat.
