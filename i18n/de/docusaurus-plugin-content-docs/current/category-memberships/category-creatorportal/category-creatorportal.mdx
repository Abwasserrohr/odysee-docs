# Kanalmitgliedschaften

Auf der Seite zu den Kanalmitgliedschaften kann ein Creator, ein Bankkonto (sofern noch nicht geschehen) und Mitgliedschaftsstufen für Unterstützer einrichten. Außerdem kann dort eingesehen werden, wie viele Menschen den Kanal unterstützen und wie viel gespendet wurde.
