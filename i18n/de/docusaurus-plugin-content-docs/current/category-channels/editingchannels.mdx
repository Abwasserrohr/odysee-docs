# Kanäle bearbeiten

Um deinen Kanal zu bearbeiten, besuche deinen Kanal und klicke auf das das Stiftsymbol unten rechts auf deinem Banner.

![](/img/de/edit_channel_arrow_light.png#light)
![](/img/de/edit_channel_arrow_dark.png#dark)

Wenn du deinen Kanal fertig bearbeitet hast, speichere alle Änderungen, damit diese übernommen werden.

![](/img/de/edit_channel_submit_arrow_light.png#light)
![](/img/de/edit_channel_submit_arrow_dark.png#dark)
