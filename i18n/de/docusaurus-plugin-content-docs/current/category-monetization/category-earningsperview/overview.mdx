# Übersicht

Mit Zuschauerprämien erhalten kreative Geistern LBC für jeden verifizierten View bzw. Ansicht. Ein verifizierter View ist ein View von einem verifizierten Nutzer. Das bedeutet, dass wir verifiziert haben, dass es sich um eine echte Person handelt.Um Missbrauch zu verhindern, werden nur die ersten sieben Views eines verifizierten Nutzers pro Tag vergütet.

## Wie viel verdiene ich pro Ansicht?

Um deine Einnahmen pro View bzw. Ansicht zu überprüfen, navigiere zur Seite Kanäle, indem du das Profilbildsymbol anklickst und im Dropdown-Menü **Kanäle** auswählst.

*image*

Deine Einnahmen pro View/Ansicht werden rechts neben deinem Kanal angezeigt.

*image*

**Hinweis: DerPreis kann sich ändern und basiert auf einer Reihe von Variablen, wie z. B. der durchschnittlichen Betrachtungsdauer, der durchschnittlichen Anzahl der Aufrufe, der Art des Inhalts, dem Engagement, dem Standort des Erstellers und dem Preis von LBC.**

## Einnahmen verfolgen

Klicken Sie auf das Profilbildsymbol in der oberen rechten Ecke und wählen Sie **Prämien**.

Auf dieser Seite werden verdiente Zuschauerprämien unter **Beanspruchte Belohnungen** als *Inhaltsansichtstipps* angezeigt.
