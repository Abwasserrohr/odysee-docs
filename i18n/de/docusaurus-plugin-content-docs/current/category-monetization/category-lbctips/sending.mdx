# LBC-Trinkgelder verschicken 

Um Trinkgeld zu verschicken, klicke auf die Schaltfläche **Unterstützen** unter einem Upload.

*image*

Diese Option ist auch für Kanäle verfügbar.

*image*

In diesem Menü wähle die Schaltfläche **Trinkgeld** mit dem LBC-Symbol aus, wähle einen Betrag und klicke auf **Trinkgeld senden**.

Dieser LBC-Tipp wird den Inhalt/Kanal verstärken bzw. boosten, bis der Ersteller ihn für die Verwendung an anderer Stelle auf Odysee freischaltet.
