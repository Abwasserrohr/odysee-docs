# LBC-Tringelder empfangen

Wenn ein LBC-Tipp beim Betrachten eines Inhalts gesendet wird, erhöht er den LBC-Anteil dieses Inhalts, bis er freigeschaltet wird. Solange er nicht freigeschaltet ist, kann er nicht an anderer Stelle auf Odysee verwendet werden.

Um einen Tipp freizuschalten, klicke auf deine **Brieftasche**. 

*image*

Hier wähle **Transaktionen** aus und ändere den **Typ** zu Empfangen.

*image*

Wenn du den Boost freischalten möchtest, klicke auf das Schloss-Symbol neben "Trinkgeld".

**Bitte beachte: Sobald ein Tipp freigeschaltet ist, wird er den Inhalt, für den er ursprünglich getippt wurde, nicht mehr boosten.

Weitere Informationen über Content-Boosts und Odysees-Einsatzsystem findest du in unserem Abschnitt über [LBC-Einsatz](https://help.odysee.tv/category-blockchain/category-staking/naming) klicken.
