# LBC-Tringelder empfangen

Wenn LBC-Trinkgeld beim Betrachten eines Inhalts gesendet wird, erhöht er den LBC-Anteil dieses Inhalts, bis er freigeschaltet wird. Solange er nicht freigeschaltet ist, kann er nicht an einer anderen Stelle auf Odysee verwendet werden.

Um Trinkgeld freizuschalten, klicke auf deine <OdyseeButton url="https://odysee.com/$/wallet"><OdyseeIcon type="credits"/> Brieftasche</OdyseeButton>. 

![](/img/de/wallet_screen_support_toggle_closed_dark.png#dark)
![](/img/de/wallet_screen_support_toggle_closed_light.png#light)

Klicke hier auf **Mehr Anzeigen <OdyseeIcon type="ChevronDown"/>** und öffne das Freischalte-Menü, indem du auf <OdyseeIcon type="Unlock" color="#de0050" /> klickst.

![](/img/de/wallet_screen_support_toggle_opened_arrow_to_unlock_dark.png#dark)
![](/img/de/wallet_screen_support_toggle_opened_arrow_to_unlock_light.png#light)


Wenn du alle Tringelder freischalten möchtest, klicke in dem geöffneten Menü auf <PrimaryOdyseeButton>Alles herauslösen</PrimaryOdyseeButton>.

![](/img/de/wallet_unlock_all_credits_modal_dark.png#dark)
![](/img/de/wallet_unlock_all_credits_modal_light.png#light)

:::info Beachte
Sobald du dein Trinkgeld freigeschaltet hast, wird der zuvor unterstützte Inhalt nicht mehr über das Boost-System unterstützt.
Weitere Informationen über Content-Boosts und Odysees-Einsatzsystem findest du in unserem Abschnitt über die **[Namensgebung](/category-blockchain/category-staking/naming)**.
:::
