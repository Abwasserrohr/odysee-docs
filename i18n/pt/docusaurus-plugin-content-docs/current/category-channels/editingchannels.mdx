# Editar um Canal

Para editar qualquer uma das configurações acima, clique no ícone de lápis no canto inferior direito do banner do seu canal.

![](https://i.imgur.com/ogPLyIt.png)

Após a edição, certifique-se de salvar as alterações.

![](https://i.imgur.com/ZISLVpY.png)
