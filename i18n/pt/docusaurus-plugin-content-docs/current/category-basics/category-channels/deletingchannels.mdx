# Deletar um Canal

Você tem algumas opções para deletar um canal.

Você pode excluí-lo na página **Canais**, usando o menu de três botões.

![](https://i.imgur.com/SLP1Axa.png)

Você também pode excluir um canal usando o menu de três botões no próprio canal. 

![](https://i.imgur.com/CNcFzOy.png)

Ou, se você decidir excluir um canal durante a edição, também há uma opção de exclusão no menu de edição.

![](https://i.imgur.com/7F8uSTA.png)
