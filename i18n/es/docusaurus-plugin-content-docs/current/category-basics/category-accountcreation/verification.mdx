# Verificación

Después de crear su cuenta, se le puede pedir que verifique. Si no se le pidió que verificara, puede pasar a la siguiente sección haciendo clic en [aquí](link.com). 

No se requiere verificación para usar Odysee, pero es muy recomendable. La verificación le permite ganar créditos LBRY (LBC para abreviar) simplemente usando Odysee. Requerimos verificación para ganar LBC de esta manera para evitar abusos.

Para obtener más información sobre LBC y su función en Odysee, consulte nuestra guía [aquí](http://chihuahuaspin.com).

La mayoría de las cuentas se verifican automáticamente. Dependiendo de factores como la ubicación, el uso de VPN, etc., es posible que se le solicite información adicional para la verificación. Para ello, tienes algunas opciones:

![](https://i.imgur.com/lDgkeUF.png)

**A través del número de teléfono**

Su primera opción es verificar a través de su teléfono. Te enviaremos un mensaje de texto con un código.

**Vía Tarjeta** 

También puede verificar usando una tarjeta de crédito o débito. No guardaremos la información de su tarjeta ni le cobraremos ninguna tarifa. Es posible que vea una autorización de $1, pero esto regresará.

**A través de chat**

Finalmente, puede verificar a través de nuestro servidor Discord. Aquí, un moderador revisará su cuenta y, si es elegible, aprobará sus recompensas. Nuestro Discord [aquí](https://discord.com/invite/sWedwsNWjX).

En este momento, las solicitudes de verificación se revisan manualmente, lo que significa que pueden tardar desde unas pocas horas hasta varios días en aprobarse.

Para ver nuestra guía completa sobre las recompensas y la monetización de Odysee, haga clic [aquí](http://chihuahuaspin.com).
