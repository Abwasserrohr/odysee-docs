# Verificación

No se requiere verificación para usar Odysee, pero es muy recomendable. La verificación le permite ganar Créditos LBRY (LBC para abreviar) simplemente usando Odysee. Requerimos verificación para ganar LBC, y de esta manera evitar abusos.

:::info Info

Para obtener más información sobre LBC y su función en Odysee, consulte nuestra guía [aquí](https://help.odysee.tv/es/category-blockchain/).

:::

La mayoría de las cuentas se verifican automáticamente. Dependiendo de factores como la ubicación, el uso de VPN, etc. Es posible que se le solicite información adicional para la verificación. Para ello, tienes algunas opciones:

**A través del número de teléfono**

Su primera opción es verificar a través de su número de teléfono. Te enviaremos un mensaje de texto con un código.

**Vía Tarjeta** 

También puede verificar usando una tarjeta de crédito o débito. No guardaremos la información de su tarjeta ni le cobraremos ninguna tarifa. Es posible que vea una autorización de pago de $1, pero esto regresará a su cuenta.

**A través de chat**

Finalmente, puede verificar a través de nuestro servidor Discord. Aquí, un moderador revisará su cuenta y, si es elegible, aprobará sus recompensas. Nuestro Discord [aquí](https://discord.com/invite/sWedwsNWjX).

En este momento, las solicitudes de verificación se revisan manualmente, lo que significa que pueden tardar desde unas pocas horas hasta varios días en aprobarse.
