# Edición de canales

Para editar cualquiera de las configuraciones anteriores, presiona el ícono del lápiz en la parte inferior derecha del banner de tu canal.

![](https://i.imgur.com/PAf8ony.png)

Después de editar, asegúrese de enviar sus cambios.

![](https://i.imgur.com/Ap0fVxC.png)
