# Crear más canales

Para crear otro canal, primero querrás volver a la página **Canales**.

![](https://i.imgur.com/A3DEdiT.png)

Desde esta página, podrá crear más canales utilizando el botón **Nuevo canal**.

![](https://i.imgur.com/xsPm12i.png)

Puede crear hasta 5 canales.
