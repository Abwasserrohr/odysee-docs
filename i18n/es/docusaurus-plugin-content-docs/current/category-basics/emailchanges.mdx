# Cambios de correo electrónico

En este momento, no tenemos una forma integrada de cambiar los correos electrónicos. Podemos cambiar los correos electrónicos, pero tenemos que hacerlo manualmente.

Si desea cambiar su correo electrónico, envíe un correo electrónico a hello@odysee.com.
