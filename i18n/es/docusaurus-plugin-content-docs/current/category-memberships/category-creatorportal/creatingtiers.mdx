# Crear niveles

El primer paso es elegir con qué canal desea interactuar. A partir de entonces, se le pedirá que configure la información de su cuenta bancaria, si aún no lo ha hecho. Para ver nuestra guía sobre cómo conectar una cuenta bancaria, haga clic [aquí](https://help.odysee.tv/es/category-basics/bankaccount).

Siguiendo la información de la cuenta bancaria, puede configurar y publicar niveles de donación.
