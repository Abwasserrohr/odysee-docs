# ¿Por qué hay límites/requisitos de sincronización?

Si dependiera de nosotros, permitiríamos que todos sincronizaran sin restricciones. Tal como está ahora, tenemos recursos limitados para sincronizaciones de canales y tenemos que asignarlos lo mejor que podamos.
