# Bienvenido al Centro de Ayuda

## ¿Qué es Odysee?

Odysee es una plataforma de medios basada en blockchain. Alojamos todo tipo de medios, como imágenes, artículos, archivos PDF, archivos de audio, etc., pero somos más conocidos por alojar videos.

:::tip

En este Centro de Ayuda puedes conocer todo lo referente a Odysee y sus funcionalidades.

:::
