# Recibiendo de propinas LBC

Cuando se envía una propina de LBC mientras se ve un contenido, aumenta el stake LBC de ese contenido hasta que se desbloquea. Hasta que no se desbloquee, no se puede usar en ningún otro lugar de Odysee.

Para desbloquear una propina, primero vaya a la billetera haciendo clic en **Su billetera**.

*imagen*

En esta pantalla, seleccionará **Transacciones** y cambiará el **Tipo** a Recibido.

*imagen*

Cuando vea el Impulso que desea reembolsar, simplemente haga clic en el icono de desbloqueo junto a "Propina" y desbloqueará la propina.

**Tenga en cuenta: una vez que se desbloquea una propina, ya no impulsará el contenido sobre el que se apoyó originalmente.**

Para obtener más información sobre mejoras de contenido y el sistema de participación de Odysee, consulte nuestra sección sobre staking haciendo clic [aquí] (https://help.odysee.tv/es/category-blockchain/category-staking/naming).
