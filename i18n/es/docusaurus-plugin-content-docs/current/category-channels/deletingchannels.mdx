# Eliminación de canales

Para eliminar un canal, tiene algunas opciones.

Puede eliminarlos en la página de **Canales**, usando el menú de tres botones.

![](https://i.imgur.com/345UtVh.png)

También puede eliminar un canal usando el menú de tres botones en el propio canal.

![](https://i.imgur.com/Bha4o9U.png)

O, si decide eliminar un canal mientras está en mitad de una edición del canal, hay una opción de eliminación en el menú de edición.

![](https://i.imgur.com/JVu5qa8.png)
